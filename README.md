# 숫자 야구 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/ent-precourse)

## 구현할 기능 목록
* 게임을 진행할 메소드 정의
* 변수 초기화 하기
* 컴퓨터의 숫자 정하기
* 컴퓨터의 숫자 카운트하기(어떤 숫자가 등장했는지)
* 숫자 입력받기
* 결과 구하기
* 결과 출력하기
* 계속할지 여부 묻기
* 테스트 코드 작성하기
