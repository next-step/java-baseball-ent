# ⚾️ 숫자 야구 게임
## ⁉️ 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 🎯 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/ent-precourse)

## ✅ 기능 목록

- 게임시작
    - 시작 문구 출력
    - 3자리 Random 수 (맞춰야되는 수) 생성
- 게임 반복 동작
    - 숫자 입력 문구 출력
        - 유저 입력에 대한 예외처리
    - 사용자로부터 숫자 입력
    - 입력된 숫자에 대한 결과 계산
    - 결과 출력
- 게임 종료 및 재시작
    - 3스트라이크 시 게임 종료 문구 출력
    - 종료 or 재시작 문구 출력
    - 종료 or 재시작 카운트 입력
        - 카운트 입력에 대한 예외 처리
